== Description ==
The AGIL plugin is here to use in any page and post of wordpress.
Use of plugin:- If you have requirement to show some files in a page,From where user can click and download that file.(Any Format).You can show it like grid of 4,5 columns based on width.
You can get all things with only one click. 

Explaination:-

You have to install this plugin.
1.Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

After installation:
In Wordpress admin, You get
1. A menu of Grid image listing.
2. In Setting option, have to give a (Mandatory)folder name in which you like to save all images     and files which has to be downloaded by the user.
3.You can set width, Height, Border color, border width of grid, Which you like to show on page.
4.You should keep name of image and file will be same like folder name.
5. you can upload a no. of folders by one click.Like.:\

You want to upload animals
so make some folders on your system.
Like zebra, Girrafe, Elephant, lion, Tiger.
and in zebra folder you have to keep image of zebra and file which you like to downded by user with the same name like zebra.(folder, file,image should be name zebra).
and then make all folders.

select all and make a zip.
and upload it in admin panel.

6. Use this shortcode where ever you like to use that grid.
   agil_image_grid.

7. To get more info.See screeshots.
